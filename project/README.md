# Proyecto Django Blog

Este proyecto es una aplicación web desarrollada con Django.

Funcionalidades:
- Listado de páginas
- Detalle de páginas
- Creación de mensajes
- Edición y eliminación de páginas
- Autenticación de usuarios (login, logout, registro)
- Panel de administración

Uso:
1. Ingresar a /pages/
2. Crear páginas desde /admin/
3. Ver detalle con "Leer más"
4. Editar y eliminar páginas
5. Enviar mensajes desde /mensaje/
